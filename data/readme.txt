The easiest way to load the sample data is to use PostMan.

1) Create a POST request to: https://sandbox.hspconsortium.org/dstu2/open-hspc-reference-api/data
2) Set headers
    Accept: application/json+fhir
    Content-Type: application/json+fhir;charset=UTF-8
